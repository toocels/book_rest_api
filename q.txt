Library Management System
You are supposed to build a “Book Directory” using Node JS. You have to create a REST API for your organization. Implement the API needed for the following user stories. Your application should be able to handle the following requests made by the user. Design a user friendly frontend using CSS skills. Write API to manage the following user requests:
GET – fetches the book details from the directory. This can include all the books or specific books by ID. 
POST – adds a new book to the directory of books. 
PUT – updates any existing book in the directory.
DELETE – delete any existing book from the directory. 


Deliverables: 


Source code for the application.
Steps to set up the server. 
API definition and how to test them.
About the source / any detailed information about the app.
